## Module Details

- **EXP 1**: Array of structures - Vaccine eligibility checker

- **EXP 4**: Writing text to files

- **EXP 6**: Number to word conversion using switch statement
- **EXP 7**: Digit frequency counter
- **EXP 8**: String sorting in lexicographical order
- **EXP 9**: Number pattern matrix generation
- **EXP 10**: Square calculation using function without arguments

### Module 9: Stack and Queue using Arrays (5 Experiments)

- **EXP 12**: Push operation on stack using array
- **EXP 13**: Display queue elements using array
- **EXP 14**: Enqueue operation on queue using array
- **EXP 15**: Dequeue operation on queue using array

- **EXP 17**: Insert node in linked list
- **EXP 18**: Traverse doubly linked list
- **EXP 19**: Insert element in doubly linked list
- **EXP 20**: Delete element from linked list

### Module 11: Functions and Arrays (5 Experiments)
- **EXP 21**: Find greatest of four numbers using function
- **EXP 22**: Maximum values for AND, OR, XOR operations
- **EXP 23**: Shelf and book query system using 2D arrays
- **EXP 24**: Calculate sum of array elements

### Module 12: Stack and Queue using Linked Lists (5 Experiments)

- Singly Linked Lists
- Doubly Linked Lists
- Pattern Programs
- Bitwise Operations
- String Manipulation
- Control Structures (switch, loops)
